# 🔐 StenoCryptX - Secure Encryption & Steganography Toolkit

## 📌 Overview
StenoCryptX is a web-based cybersecurity tool that combines AES-256 GCM encryption with image steganography to enable secure and hidden communication.

## 🔥 Features
- AES-256 GCM encryption
- Image steganography (LSB technique)
- Fully client-side processing (no server)
- Secure file & text encryption
- Angular-based modern UI

## ⚙️ Tech Stack
- Angular
- TypeScript
- Web Crypto API
- HTML5 Canvas
